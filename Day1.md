## 来吧，任何事情的第一步都是，布置环境。

AI跟我说SLAM领域都是在Linux (Ubuntu) 环境下运行得最顺畅，直接装一个**WSL2 (Windows Subsystem for Linux)**，在windows里面跑Linux吧，还能调用你的4060显卡呢！比虚拟机强多了！

管理员身份运行Windows PowerShell，输入：  
`wsl --install`  
这里是开着梯子代理下的，不然可能会遇到网络问题，最后下载好了。  
提示说这是“适用于 Linux 的 Windows 子系统 2.6.1”，行，我看不懂，略过。  
最后看到  
`password updated successfully`  
知道下好了，继续跟着AI走。输入：  
`cat /etc/os-release && sudo apt update`  
创建账户、密码等，然后得知自己的版本是**Ubuntu 24.04.3 LTS**，但gemini说最好走“经典稳健”路线（换成 Ubuntu 20.04）  遂使用：  
`wsl --unregister Ubuntu`  
卸载了ubuntu，又打开微软商店，和网络问题做了n次斗争之后下载了**Ubuntu 20.04.6 LTS**  
在windows的开始菜单里点开**Ubuntu 20.04.6 LTS**看到完全不一样的Ubuntu命令行窗口的时候，这个在Windows系统下的Linux系统也就配置好了。  

## 继续配置ROS环境  
AI指导说安装**ROS Noetic**，这是 Ubuntu 20.04 对应的 ROS 版本。  
先更新一遍系统，然后安装一个叫**wget**的下载工具：  
`sudo apt update && sudo apt install wget -y`  
过程中系统会提示你输入密码，输入创建系统时自己设置的密码并回车，输入时屏幕上不会有反应，这是正常的。  
然后自己配置**Noetic**会比较麻烦，让我使用国内大神的一键配置环境：  
`wget http://fishros.com/install -O fishros && . fishros`  
选择“**一键安装**”、“**更换系统源再继续安装**”、“**oetic(ROS1)**”、“**desktop-full [推荐]**”等等，  
反正最后安装完了，到此，整个Linux环境下的Ros环境也就配置完毕了！  
之后只需要在windows的开始菜单里打开Ubuntu20.04就可以启动这个你自己的Linux了。

## 开始ROS界的“Hello,World!”---召唤小海龟（Turtlesim）  
遂了解到ROS 的一个核心概念是多节点协同，简单来说就是需要多个命令窗口一起工作。Gemini称为**召唤海龟三步大法**。  
### 第一步，召唤大脑(Master)
先用打开的第一个命令窗口运行：  
`roscore`回车  
我也看不懂，但是看英文的话应该是个ros的核心什么玩意之类的，反正AI让我按完回车后就别管，继续加新的ubuntu标签页就完了，行，照做吧。  
gemini对此的描述是“**这是 ROS 的“中枢神经”，它必须时刻运行，其他节点才能互相说话**”  
### 第二步，召唤躯干(Node)
在命令行顶部标签栏的右侧点击 + 号旁边的小箭头选择 Ubuntu，继续输入：  
`rosrun turtlesim turtlesim_node`回车  
屏幕上弹出来了一个蓝色的图形化小窗口，中间有一只静止的小海龟。嗯，点击没反应，方向键也没反应，我的评价是，只要不报错，多丑我都能接受。  
至于代码是什么意思，我说实话只能看懂ros和run的结合，运行了什么东西的意思。
### 第三步，连接控制器（Teleop）
继续打开第三个命令窗口  
输入：  
`rosrun turtlesim turtle_teleop_key`回车  
屏幕显示Reading from keyboard，让我确保鼠标焦点在这个第 3 个黑色的终端，然后按下键盘的方向键（上下左右）。  
海龟动起来了，哈哈。
